# Dr. Mohamad Izani — Personal Website

A clean, modern academic website. Plain HTML/CSS/JS — no build step, no dependencies. The header and footer are generated by one shared script, so navigation stays consistent everywhere and is edited in a single place.

## Pages

```
index.html               About / academic profile
publications.html        Journal articles, books & chapters, conference papers (filterable)
portfolio-persona.html   Portfolio ▸ My Work (creative & research portfolio)
portfolio-students.html  Portfolio ▸ Student Work (mentorship showcase)
projects.html            Projects, research grants, courses taught
training.html            GenAI Training (AI4LIFE) + testimonials
awards.html              Achievements & Awards
blog.html                Blog index
  blog-genai-education.html    Post 1 — Teaching in the Age of Generative AI
  blog-digital-heritage.html   Post 2 — Preserving Heritage in Pixels
  blog-2d-to-3d-ai.html        Post 3 — From 2D to 3D
contact.html             Contact details, profiles, message form
assets/
  style.css              Theme (navy / slate, serif headings)
  main.js                Header + footer, dropdown nav, animations, filter, lightbox
  profile.jpg            Your headshot (from your CV)
  gallery/               Drop travel & portfolio photos here
```

## Preview locally

Double-click `index.html`. Every page links through the shared navigation.

## Editing common things

**Navigation** — edit the `NAV` list at the top of `assets/main.js`. Changes apply to every page at once.

**Add a photo** (travel / portfolio) — put the image in `assets/gallery/`, then in the relevant page replace a tile's `<div class="ph">…</div>` with `<img src="assets/gallery/your-file.jpg" alt="…">`. Photos open in a lightbox.

**Testimonials** — the three quotes on `training.html` are samples. Replace the text, initials and roles with real feedback you've collected.

**Blog** — three starter posts are included; edit them freely. To add a post, copy any `blog-*.html` file, write your article, and add a matching card in `blog.html`.

## Publish to www.izanizainal.com

The site is static, so any host works:

- **Web host (cPanel / FTP):** upload the **contents** of this folder into `public_html` (or `www`) so `index.html` sits at the root.
- **Netlify:** app.netlify.com → *Add new site* → *Deploy manually* → drag this folder in → add your custom domain.
- **GitHub Pages:** push these files to a repo, enable Pages, add `izanizainal.com` as the custom domain with a `CNAME` DNS record.

## Notes

- Fonts (Source Serif 4 + Inter) load from Google Fonts, so full typography needs an internet connection; it falls back to system fonts offline.
- The contact form opens the visitor's email app with the message pre-filled — no server required.
- Content is drawn from your 2025 CV. The publications page is a curated selection with a link to your full Google Scholar record.
